# This is the first cycle - Cycle 01: Laptop Price Analyst

- Predict laptop price in Indian market. I get the data from the url:"<https://raw.githubusercontent.com/campusx-official/laptop-price-predictor-regression-project/main/laptop_data.csv>"
- In this project, i use Random forest to predict laptop price base on the data of the new laptop that user choose.
  I use Random forest because Linear Regression is too simple, easily make huge error if there are some outliers (rare brand laptop, ...); Neural Networks (DL) is too complicated for 1303 line of data; Random forest is the 'sweet point'.
- The system can process raw text/data, calculate PPI, reach MAE ~ 3 million VND
